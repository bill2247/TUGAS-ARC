Merge adalah proses menyatukan dua branch. Katakan kita menyatukan branch A dan B. Pada proses ini, setiap file yang ada di branch A dan B akan disatukan sehingga dihasilkan branch A dan B keduanya berisi gabungan file tersebut. Lalu bagaimana jika ketika penggabungan ada beberapa file yang memiliki line yang berbeda isinya? hal tersebut dinamakan merge conflict. MIsalkan sekarang kita berada di branch A, yang kita lakukan untuk menghadapi merge conflict adalah menuliskan "$ git merge B" pada git bash, lalu secara otomatis akan diarahkan ke code editor yang kita pilih saat instalasi git. Disitu ditampilkan conflict yang terjadi pada kedua branch. Sekarang Tugas kita membenarkan conflict yang terjadi. Jika sudah save lalu close code editor, lalu tuliskan "git commit -am"masukkan pesan commit"" pada git bash. Selesai
